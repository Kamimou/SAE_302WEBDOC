# Canteen WebDoc

## Project Overview
Canteen WebDoc is a web project designed to provide an interactive experience centered around canteen ingredients. The homepage features a visually appealing plate that showcases various ingredients, which animate on hover and are clickable for more information.

## Features
- Interactive plate component displaying ingredients.
- Ingredients change color and animate on hover.
- Clickable ingredients that provide additional information.
- Responsive design suitable for various devices.

## Color Palette
- Light Green
- Olive Green
- Off-White
- Light Beige
- Pale Yellow
- Soft Gray
- Touches of Tomato Red or Orange

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the necessary dependencies using npm:
   ```
   npm install
   ```
4. Open `src/index.html` in your web browser to view the project.

## Usage Guidelines
- Hover over the ingredients on the plate to see the animations.
- Click on an ingredient to learn more about it.
